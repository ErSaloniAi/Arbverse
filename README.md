# 🚀 ARBVERSE – Web3 Learning & Crypto Dashboard

A modern, responsive **3-page Web3 educational website** built using **HTML, CSS, and JavaScript**.

ARBVERSE helps users understand **Blockchain fundamentals**, **Arbitrum Layer 2**, and track **live cryptocurrency prices** in real time.

# 📄 Pages

## 🏠 1. Home Page

Learn about **Arbitrum Layer 2** and why Ethereum needed scaling.

### Includes:

- ⚡ Hero Section
- ⛓️ Layer 2 Introduction
- 💸 Gas Fee Problem
- 🚀 Arbitrum Benefits
- 🔐 Ethereum Security

---

## 📚 2. Concepts Page

Understand core Web3 concepts with visual comparison cards.

### Topics Covered:

- 🌍 Web2 vs Web3
- ₿ Bitcoin vs Ethereum
- 🔑 Public Key vs Private Key
- 🗄️ Blockchain vs Traditional Database

Track real-time cryptocurrency prices.

### Features:

- ₿ Bitcoin Live Price
- ◇ Ethereum Live Price
